# 🏏 IPL Data Analysis

## 📌 Project Overview

This project analyzes IPL match data using Python. The goal is to discover useful insights about teams, players, and matches through data analysis and visualization.

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Analysis Performed

- Top 10 Run Scorers
- Top 10 Wicket Takers
- Orange Cap Analysis
- Purple Cap Analysis
- Best Batting Performance
- Best Bowling Performance
- Team with Most IPL Titles
- Toss Decision Analysis
- Top IPL Host Cities
- Top Fielders

---

## 📂 Dataset

- matches.csv
- deliveries.csv

---

## 🚀 How to Run

1. Download this repository.
2. Install the required libraries:
   ```
   pip install -r requirements.txt
   ```
3. Open `IPL_Data_Analysis.ipynb` in Jupyter Notebook.
